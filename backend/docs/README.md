# docs/

Esta pasta concentra os documentos do projeto (entrega e apoio).

- A aplicação expõe uma página pública de listagem em: `/documentos`
- Os arquivos desta pasta são servidos pelo back-end em: `/docs/<arquivo>`

## Como adicionar novos documentos

Basta inserir aqui arquivos em um dos formatos suportados:
- `.pdf`
- `.md`
- `.txt`

A listagem é automática.
